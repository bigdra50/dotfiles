## AI臭さを除去する文章作成

### 避けるべき表現パターン

#### 機械的な装飾
- 絵文字による装飾的なマーク（✅、🔥、⚡など）
- 過度な太字・斜体の使用
- 機械的なリスト書式

#### 誇張表現
- 革新的、画期的、劇的な変化
- 最高級、最先端、究極の
- 完璧な、理想的な、素晴らしい

#### 不自然な強調
- 感嘆符の多用
- 「**非常に**重要」のような強調の濫用
- 同一語句の反復

#### AI特有の文体
- 「以下の通りです：」のような機械的な導入
- 「〜について説明します」の多用
- 「まとめると」「結論として」の定型句

### 推奨する文章スタイル

#### 簡潔性
- 一文一義を心がける
- 修飾語は最小限に抑える
- 冗長な表現を削除する

#### 自然さ
- 能動態を基本とする
- 具体的な表現を選ぶ
- 読み手の立場で書く

#### 明確性
- 曖昧な表現を避ける
- 専門用語は適切に説明する
- 構造を明確にする

### 文章チェック項目

- [ ] 装飾的な絵文字を使用していない
- [ ] 誇張表現を避けている
- [ ] 不自然な強調を削除している
- [ ] 機械的な定型句を使用していない
- [ ] 一文が適切な長さに収まっている
- [ ] 読み手にとって自然な文章になっている
