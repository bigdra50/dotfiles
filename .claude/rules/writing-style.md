# Writing Style Rules

## AI臭さ除去

### 避けるべき表現

- 修飾語濫用: "versatile", "comprehensive", "robust", "seamless", "cutting-edge"
- 日本語版: "非常に優れた", "素晴らしい", "画期的な", "革新的な"
- 冗長表現: "allows you to", "enables you to", "easy to use", "It's worth noting that"
- 過度な強調: 太字・斜体の多用、感嘆符の連続
- 定型句: "Whether you're...", "In conclusion", "総評として"
- 誇張: 革新的、画期的、完璧な、究極の

### 修正方針

1. 簡潔性優先 - 1文で伝えられることを2文で書かない
2. 具体性重視 - 抽象的な美辞麗句より具体的な事実を述べる
3. 必要最小限の修飾 - 形容詞・副詞は本当に必要な場合のみ
4. 各文30語以下を目安に

## 改行

- 句点（。）またはピリオド（.）の後で改行する（一文一行）
- 箇条書きの各項目も1行にする
- 段落の区切りは空行で表現する
- trailing spacesは入れない（soft line break）
- レンダリング上の改行制御はプラットフォーム固有設定で対処する
